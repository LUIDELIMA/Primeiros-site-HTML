Tag html

(Volta)

A tag html, é considerada como elemento básico da estrutura do arquivo HTML. Dentro dessa tag estará todos os elementos do seu arquivo HTML.

Vale lembrar que todo arquivo HTML deve iniciar-se com essa tag e finalizar com a mesma.

Tag head

(Volta)

Já nessa tag HEAD, é onde e definido o cabeçalho o documento, ele não possui nenhum valor visível, mas é capaz de transmitir ao navegador diversas informações úteis e essencial para apresentação do seu documento HTML.

Tag title

(Volta)

Na teg TITLE, é onde é definido o título da páginaque aparecerá na abas da janela do navegador.

Tag body

(Volta)

Para finalizar, a tag BODY e a que representa o corpo do documento. Em síntese, é nessa tag que todos os elementos visíveis do seu site devem ser inseridos, E onde será visualizada pelo usuário.
